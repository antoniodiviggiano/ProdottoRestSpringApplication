# Spring Restful web services CRUD

Link utili:

[Postman](https://www.postman.com/), [Docker](https://www.docker.com/) 

### CRUD

- GET :
```
http://localhost:8080/prodotti
```
- GET: 
```
http://localhost:8080/prodotto/{id}
```
- POST:
```
http://localhost:8080/inserisciprodotto
```
- PUT:
```
http://localhost:8080/prodotto/{id}
```
